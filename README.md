## Installation

### Docker
```bash
git clone https://github.com/PashaHatsune/MxUserbot.git
cd MxUserbot
docker-compose up -d --build
```

### Manual Installation
```bash
git clone https://github.com/PashaHatsune/MxUserbot.git
cd MxUserbot
uv sync
uv run -m src.mxuserbot
```
